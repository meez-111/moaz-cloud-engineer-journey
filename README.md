## 🇸🇦 KSA Compliance Focus

This journey stands out by integrating Saudi Arabian regulations from early stages:

- **PDPL** (Personal Data Protection Law): Data localization, consent management, breach notification, retention policies
- **NCA** (National Cybersecurity Authority): Essential Cybersecurity Controls (ECC) and Critical Cybersecurity Controls (CCC)
- **Regional Best Practices**: Preference for `me-south-1` Riyadh region, data encryption at rest and in transit, audit logging
- **Practical Implementation**: Region compliance checker script, PDPL compliance guide, and ECC mapping to AWS services

Full compliance documentation and scripts → [ksa-compliance/](ksa-compliance/)
